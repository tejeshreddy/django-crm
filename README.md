# django-crm
A Django CRM to maintain customer info.

## Commands to Run

- To run the server <br>
```bash
python3 manage.py runserver
```

- To make migrations <br>

```bash
python3 manage.py makemigrations
```

- To migrate the changes performed <br>

```bash
python3 manage.py migrate
```
